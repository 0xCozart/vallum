# Team and Maintainer Credibility

IOTA GasKit is maintained and built by 0xCozart as an open-source infrastructure toolkit for IOTA sponsored-transaction adoption.

Relevant delivery experience shown in this repository includes full-stack TypeScript and Node service implementation, SDK design, policy and quota enforcement, deterministic test automation, package dry-run readiness, security documentation, secret-hygiene checks, and operator-focused deployment documentation.

Current proof already completed:

- deterministic local gateway, SDK, example, and demo-dApp checks;
- authenticated policy simulation and fail-closed allowlist behavior;
- sanitized decision events, in-memory usage views, and file-backed JSONL usage-event foundation;
- package publication readiness checks through `npm run pack:check`;
- tracked-file secret scanning through `npm run secrets:scan`;
- documented public IOTA testnet sponsored execute evidence.

Delivery capacity is a focused maintainer-led implementation track. The proposed milestones are scoped as reviewable open-source increments, not as a private hosted-service launch.
