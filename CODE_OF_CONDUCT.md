# Contributor Covenant Code of Conduct

## Our pledge

IOTA GasKit is intended to be useful public infrastructure for IOTA builders. We pledge to make participation in this project respectful, welcoming, and focused on technical quality, safety, and ecosystem benefit.

## Expected behavior

- Be respectful and constructive in issues, pull requests, reviews, and discussions.
- Assume good intent while still being precise about security, reliability, and documentation concerns.
- Keep sponsor-wallet safety, user safety, and open-source maintainability ahead of personal preference.
- Report suspected security issues through `SECURITY.md` instead of public issue threads.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private keys, mnemonics, API keys, bearer tokens, or other secrets.
- Encouraging unsafe sponsor-wallet operations or intentionally bypassing policy controls.
- Spam, impersonation, or intentionally disruptive behavior.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct. Maintainers may also temporarily or permanently restrict participation for repeated or severe violations.

For conduct concerns, contact the maintainers through the repository owner account or the private reporting path listed in `SECURITY.md` when the concern involves sensitive information.

## Attribution

This code of conduct is adapted from the Contributor Covenant style of community standards and tailored for this project’s security-sensitive sponsored-transaction scope.
